CodeIgniter Twig and HMVC Integration for Code Igniter 3.1.10

## Server Requirements
PHP version 7 or newer.
Twig 2.6.2 or later.

## Folder Structure
```
root/
└── application/
    └── module/
        └── welcome/
            └── controllers
                └── Welcome.php
            └── models
                └── Model.php
            └── views
                └── View.twig
```
